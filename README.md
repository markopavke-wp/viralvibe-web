# Viral Vibe (Next.js)

Modern rewrite of viralvibe.rs with glass UI, motion, and EN/SR locales.

## Run

```bash
cd viralvibe-web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — redirects to `/en`.  
Serbian: [http://localhost:3000/sr](http://localhost:3000/sr)

## Stack

- Next.js App Router
- TypeScript + Tailwind CSS v4
- Framer Motion
- Locale routes: `/en/*` and `/sr/*`
